# WARP.md

This file provides guidance to WARP (warp.dev) when working with code in this repository.

## Project Overview

This repository explores what enables in-context learning in transformers through analysis of attention mechanisms. This is a research-oriented codebase focused on understanding the underlying mechanisms that allow transformers to learn from context.

## Current Status

This is a new repository currently in the planning/initialization phase. The codebase will likely include:
- Transformer model implementations or modifications for attention analysis
- Experiments analyzing attention patterns during in-context learning
- Visualization and analysis tools for attention mechanisms
- Data processing pipelines for in-context learning tasks

## Development Guidelines

When code is added to this repository:
- This is a research project focused on attention mechanisms and in-context learning
- Prioritize clarity and reproducibility in experimental code
- Include clear documentation of experimental setups and hyperparameters
- When implementing attention analysis tools, ensure visualizations are interpretable
